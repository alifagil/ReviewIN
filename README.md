# 🚀 ReviewIN – Fake Review Detection System

ReviewIN is a web-based Fake Review Detection System developed to help users identify potentially fraudulent product reviews on e-commerce platforms.

The system utilizes Natural Language Processing (NLP), Machine Learning, and Explainable AI techniques to analyze review credibility, classify reviews as **BOT** or **HUMAN**, generate a **Fraud Score**, and provide insights into suspicious review patterns.

# 📖 Project Overview

Online reviews play a significant role in influencing purchasing decisions. However, the increasing number of fake reviews generated by bots, click farms, and coordinated manipulation campaigns has made it difficult for consumers to determine review authenticity.

ReviewIN addresses this challenge through an automated review analysis platform that combines:

* Web Scraping
* Regex-Based Text Processing
* Machine Learning Classification
* Explainable AI
* Interactive Visualization

The system aims to improve transparency and support safer purchasing decisions.

---

# ✨ Key Features

## 🔗 Amazon URL Analysis

Users can submit an Amazon product URL.

The system automatically:

* Scrapes reviews using Selenium
* Extracts review content
* Processes the collected dataset
* Generates Fraud Analysis results

---

## 📝 Manual Review Analysis

Users can paste review text directly into the system.

This feature is useful when:

* Scraping is unavailable
* Users already have exported review data
* Bulk review analysis is required

---

## 🧹 Regex-Based Review Segmentation

A custom Regex Engine is implemented to:

* Split bulk review text into individual reviews
* Remove unnecessary metadata
* Clean Amazon-specific formatting
* Improve model prediction consistency

Examples of removed metadata include:

* Ratings
* Dates
* Locations
* Verified Purchase badges
* Product variations

---

## 🤖 Machine Learning Classification

Each review is analyzed and classified into:

* HUMAN Review
* BOT Review

using a trained Logistic Regression model.

---

## 📊 Fraud Score Generation

The system calculates:

```text
Fraud Score =
(Number of BOT Reviews / Total Reviews) × 100%
```

This score provides a quick assessment of review credibility.

---

## 🔍 Explainable AI Indicators

ReviewIN includes a lightweight Explainable AI module.

The system identifies suspicious keywords that contribute significantly to BOT predictions using Logistic Regression coefficient weights.

This allows users to understand why a review is considered suspicious.

---

## 📈 Interactive Dashboard

Analysis results are displayed through:

* Fraud Score
* BOT Percentage
* HUMAN Percentage
* Chart.js Visualizations
* Classification Tables
* Fraud Indicators

# 🧠 Machine Learning Pipeline

```text
Review Input
      ↓
Regex Cleaning
      ↓
TF-IDF Vectorization
      ↓
Logistic Regression Prediction
      ↓
Fraud Score Calculation
      ↓
Dashboard Visualization
```

### TF-IDF

Converts textual reviews into numerical feature vectors based on word importance.

### Logistic Regression

Performs binary classification:

* 0 → HUMAN
* 1 → BOT

### Explainable AI

Extracts influential keywords responsible for suspicious predictions.
